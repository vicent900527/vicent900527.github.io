### 操作
---

运行工程			``` ctrl + R ```

拉取Git			``` command + T ```

提交Git			``` command + K ```

代码优化			``` ctrl + alt + O ```

### 光标
---

移动到行首		``` ctrl + A ```

移动到行尾		``` ctrl + E ```

复制整行			``` command + D ```

删除整行			``` command + delete ```

前进				``` ctrl + F ```

后退				``` ctrl + B ```

上一行			``` ctrl + P ```

下一行			``` ctrl + N ```


### 后悔
---

后退				``` command + Z ```

前进				``` command + shift + Z ```

### 搜索
---

文件内容搜索		``` command + F ```

工程内容搜索		``` command + shift + F ```

类搜索			``` command + O ```

文件搜索 			``` command + shift + O ```

最近使用文件搜索 	``` 双击 shift ```
